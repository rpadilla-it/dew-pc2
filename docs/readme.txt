Pasos para correr el proyecto:


1. Tener instalado el JDK java y IDE Netbeans 7.X 
2. Restaurar la base de datos del archivo  db-sunat-sql.sql
3. Cambiar los valores de conexion a la Base de datos para el proyecto en:
   * WebPages/WEB-INF/context/spring-base.xml
4. Cambiar los valores de conexion a la Base de datos para ejecutar los test en:
   * Other Test Sources/src/test/resources/default/spring-base-test.xml
